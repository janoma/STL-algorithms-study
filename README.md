# Study of STL's <algorithm> header
To the point: examples to study the algorithms in `C++`'s `<algorithm>` header.

## Installation
```bash
$ git clone https://github.com/janoma/stl-algorithm-study.git
$ cd stl-algorithm-study
$ cmake .
$ make
$ make test
```
